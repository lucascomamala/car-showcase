## 🛠️ Installation

Use the package manager [npm](https://www.npmjs.com/) to install ReactTube.

```bash
npm install
```

## 🖥️ Usage

To run the project locally:

```bash
npm run dev
```

To build the project:

```bash
npm run build
```
